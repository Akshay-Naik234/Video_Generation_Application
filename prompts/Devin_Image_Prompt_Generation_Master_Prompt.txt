- Vary durations naturally (4-12 seconds) — faster cuts for intense moments, longer holds for emotional beats
- PACING BY SECTION (for 12-minute video):

  - Early Life (0:30-2:00): MEDIUM pace, 6-8 seconds per image (nostalgia, warmth)

- For exposition (dates, facts, context), use establishing shots and wider compositions

VISUAL TRANSITION RULES (CRITICAL — prevents jarring cuts):
Every consecutive pair of images must have a clear visual transition logic. Use these transition types:

- **ZOOM IN**: Wide → Medium → Close-up → Extreme close-up (progressively tighter framing)
- **ZOOM OUT**: Extreme close-up → Close-up → Medium → Wide (progressively wider framing)
- **LOCATION SHIFT**: Same framing type but different location (interior→exterior, room→street, city→countryside)
- **TIME SHIFT**: Same location but different time period (young→old, day→night, past→present)
- **SCALE SHIFT**: Dramatic jump between intimate and epic scale (hands→aerial, face→crowd)
- **COLOR SHIFT**: Same scene type but shifting color temperature (warm→cool at section transitions)
- **SUBJECT SHIFT**: Shift from the subject to someone else (protagonist→antagonist, person→crowd)
- **OBJECT BRIDGE**: A significant object appears in consecutive images from different angles/contexts (a letter being written → the same letter being read)

RULES:
- NEVER place two images with the same framing AND the same location back-to-back
- At SECTION TRANSITIONS (e.g., Early Life → The Spark), use a dramatic transition: color shift + location shift
- Within a section, alternate between at least 3 different transition types
- Every 10 images, include at least ONE dramatic scale shift (intimate → epic or vice versa)

- CTA: Clean, neutral, channel-branded feel

SHOT VARIETY RULES (CRITICAL FOR RETENTION):
Based on analysis of viral biography channels, visual monotony is the #1 cause of viewer drop-off.
- NEVER use the same shot type twice in a row (e.g., no two consecutive close-ups)
- Follow this shot rotation pattern as a guideline:
  wide -> close-up -> detail -> medium -> over-shoulder -> wide -> extreme close-up -> landscape -> etc.
- Every 5 images, include at least ONE of these high-impact shot types:
  - Extreme close-up of hands, eyes, or a significant object
  - Silhouette against dramatic backlight
  - Aerial/bird's eye view of a location
  - Over-shoulder shot implying someone watching
  - Still-life arrangement of era-appropriate objects
- For MONTAGE sections (rise, conflict), use faster visual variety with more shot types
- For EMOTIONAL sections (fall, legacy), use repetitive close-ups and medium shots for intimacy
